# 🧮 Modern Python Calculator

โปรเจกต์เครื่องคิดเลขดีไซน์ทันสมัย (Modern Dark Mode) เขียนด้วยภาษา Python โดยใช้ไลบรารีมาตรฐาน `tkinter` (ไม่ต้องติดตั้งโมดูลเพิ่มเติม) โปรเจกต์นี้สร้างขึ้นเพื่อทดสอบการเขียน GUI เบื้องต้นและการอัปโหลดโค้ดขึ้น GitHub

## ✨ ฟีเจอร์หลัก (Features)

*   **การคำนวณพื้นฐาน:** รองรับการ บวก (+), ลบ (-), คูณ (*), หาร (/)
*   **ฟังก์ชันเพิ่มเติม:** รองรับการหาเปอร์เซ็นต์ (%)
*   **การจัดการข้อมูล:** มีปุ่มลบตัวเลขทีละหลัก (DEL) และปุ่มล้างค่าทั้งหมด (AC)
*   **UI/UX:** ออกแบบด้วยโทนสีเข้ม (Dark Theme) สบายตา ปุ่มกดขนาดใหญ่ใช้งานง่าย
*   **ป้องกันข้อผิดพลาด:** จัดการ Error ขั้นพื้นฐาน (เช่น การหารด้วยศูนย์ หรือสมการที่ไม่ถูกต้อง)

## 🛠️ สิ่งที่ระบบต้องการ (Prerequisites)

*   คอมพิวเตอร์ที่ติดตั้ง **Python 3.x** ขึ้นไป (สามารถดาวน์โหลดได้ที่ [python.org](https://www.python.org/downloads/))
